## Installation

Follow these steps to set up the project locally:

```bash

git clone https://github.com/your-username/cryptocurrency-tracker.git

# Navigate to the project directory
cd cryptocurrency-tracker

# Install dependencies
yarn install

# Run the project
yarn start
